# Medium Task: Multi-Modal VAE for Music Clustering

**Course:** Neural Networks  
**Author:** Moin Mostakim  
**Submission:** January 10th, 2026

## Overview

This project implements a multi-modal Variational Autoencoder (VAE) system that combines audio features and lyrics embeddings for unsupervised music genre clustering. The system compares multiple fusion strategies and clustering algorithms.

## Features

- **Dual VAE Architecture**: Convolutional VAE for audio + Dense VAE for lyrics
- **Multi-Modal Fusion**: 6 different fusion strategies
- **3 Clustering Algorithms**: K-Means, Agglomerative, DBSCAN
- **6 Evaluation Metrics**: Complete clustering quality assessment
- **Comprehensive Visualizations**: t-SNE plots, heatmaps, comparison charts
